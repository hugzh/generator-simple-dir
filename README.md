# generator-simple-dir
一个非常小的yeoman-generotor，
用于自动生成PC端或者移动端页面模板.
## 目录结构
```
|__ components
    |__ comp_demo
        |__ comp_demo.tpl
        |__ index.js
        |__ comp_demo.css
|__ pages
    |__ page_demo
        |__ page_demo.html
        |__ page_demo.css
        |__ page_demo.js
|__ static
    |__ css
    |__ js
    |__ image
```
